# Security Policy

## Responsible Use

LLMbreaker is a security research tool designed for **authorized testing only**. It is intended to help organizations discover and mitigate vulnerabilities in their own LLM applications. Using this tool against systems without explicit permission is illegal and unethical.

By using LLMbreaker, you agree to:

1.  Only test systems you own or have explicit written permission to test.
2.  Report any vulnerabilities you discover responsibly to the system owner.
3.  Not use this tool for malicious purposes.

## Reporting a Vulnerability in LLMbreaker Itself

If you discover a security vulnerability within LLMbreaker itself, please do **not** open a public GitHub issue. Instead, please report it responsibly by emailing the maintainers directly.

We will acknowledge your report within 48 hours and will work with you to understand and address the issue.

## Disclosure Policy

When we receive a security bug report, we will:

1.  Confirm the problem and determine the affected versions.
2.  Audit code to find any potential similar problems.
3.  Prepare fixes for all releases still under maintenance.
4.  Release new security fix versions as soon as possible.
